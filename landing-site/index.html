Features

5 layers of policy enforcement

Every agent execution is checked against 5 policy layers in milliseconds.

1. Tool ACL

Define which tools each agent can access. web_search, browser, mcp_execute, code_run, multi_agent.

2. Daily Quotas

Max runs per day per agent. Prevents runaway executions.

3. Monthly Token Metering

Track model token consumption per agent per month. Auto-decline when exceeded.

4. Concurrent Limits

Max simultaneous agent sessions. Track start/stop in real time.

5. Rate Limiting

Sliding window rate limit per agent. Configurable RPM.

+ 6. Revocation

Instantly disable any agent. Full audit trail. Enterprise compliance.
